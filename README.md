# 十一的一年级语文

面向中国大陆一年级小学生和家长的语文同步互动学习网站。项目以人教版小学语文一年级教材体系为参考，重点围绕 2024 版一年级语文学习内容，提供课程地图、识字、拼音、阅读、写字、口语交际和家长中心等功能。

这个项目不是教材全文复制站，而是一个家庭场景下的“课本同步互动学习工具”：孩子每天用 10 到 15 分钟完成一小组任务，家长可以看到进度、导入本地课文、陪伴朗读和确认练习结果。项目默认不上传儿童姓名、录音或学习记录。

## 项目背景

中国大陆小学一年级语文学习有几个很现实的家庭场景：

- 孩子刚进入小学，需要同时适应拼音、识字、朗读、写字和口语表达。
- 家长希望跟着课本节奏复习，但不想把练习变成额外压力。
- 低龄孩子使用网站时，需要大按钮、少文字、明确反馈和短时间任务。
- 朗读录音、学习记录、孩子姓名等数据应尽量留在家庭设备内。
- 公开开源项目不应该直接收录教材 PDF、教材插图或完整课文原文。

因此，本项目选择做“结构同步、练习互动、数据本地保存”的学习网站，而不是公开复制教材内容。

## 教材与内容说明

项目以人教版小学语文一年级教材结构为参考，重点整理 2024 版一年级语文学习体系，并预置一年级上册、下册课程目录、学习目标、生字、词语和练习模板。

为了保护版权，仓库内不包含：

- 教材 PDF 文件
- 教材插图
- 整篇课文原文
- 可替代正版教材阅读的完整内容

仓库内只保留：

- 课程目录
- 单元与课文结构
- 学习目标
- 会认字、会写字
- 少量关键词语
- 互动练习模板

完整课文可以由家长在阅读页本地导入，导入内容只保存在当前浏览器，不上传到服务器，也不会进入 GitHub 仓库。

## 功能亮点

- 首页：今日学习、继续上次、拼音、识字、阅读、写字、口语、家长中心入口
- 课程地图：一年级上册、下册完整课程目录，按单元查看学习内容
- 课时页：每课展示“今天学什么”“我会认”“我会写”“我来练”
- 拼音乐园：拼音卡、拼读配对、拼读小火车
- 识字花园：按册、单元、课文筛选生字卡
- 写字练习：静态田字格、拼音、组词、家长确认
- 课文阅读：本地课文导入、录音与回放、家长确认
- 口语交际：真实生活场景卡片和家长观察清单
- 家长中心：学习记录、复习队列、JSON 导入导出、重置本地数据
- 本地优先：默认不依赖云端账号，不做排名，不制造学习焦虑

## 设计原则

- 每次学习 10 到 15 分钟，任务短、反馈明确。
- 面向低龄孩子，界面尽量清楚、稳定、少干扰。
- 以鼓励为主，不使用“失败”“落后”等焦虑化文案。
- 家长参与确认，避免自动语音评分或自动打分带来的误判。
- 儿童数据本地保存，不接入广告、支付、第三方追踪或云端账号系统。
- 尊重教材版权，公开仓库不收录完整课文和教材图片。

## iPad 局域网访问

如果笔记本和 iPad 在同一个 Wi-Fi，可以让笔记本作为本地服务器：

```bash
npm run dev -- --hostname 0.0.0.0
```

查看笔记本局域网 IP：

```bash
ipconfig getifaddr en0
```

然后在 iPad 浏览器打开：

```text
http://你的笔记本IP:3000
```

本地服务器模式下，学习进度会优先写入笔记本电脑的：

```text
data/progress.json
```

这个文件不会提交到 GitHub。如果服务器不可用，前端会退回浏览器本地保存。

## GitHub Pages 部署

推送到 `main` 分支后，GitHub Actions 会自动运行测试、类型检查和静态构建，并发布到 GitHub Pages。

首次使用时，在 GitHub 仓库的 `Settings` -> `Pages` 中，将 `Build and deployment` 的 `Source` 设为 `GitHub Actions`。部署完成后访问：

```text
https://jackli01030.github.io/shiyi-grade-one-yuwen/
```

GitHub Pages 是静态站点，不提供 `/api/progress` 服务。线上学习进度会保存在浏览器本地；本地开发服务器仍可使用 `data/progress.json`。

工作流发布时会在 GitHub Actions 的临时构建目录里移除 `app/api`，只导出静态页面，不会改动仓库源码。

## 数据与隐私

- 学习记录保存在浏览器本地，或在家庭局域网模式下保存在笔记本本地文件。
- 录音只用于本机回放。
- 不上传儿童姓名、录音或学习记录。
- 不包含后端数据库和云端账号系统。
- 不接入广告、支付或第三方追踪。
- `data/progress.json` 已加入 `.gitignore`，不会提交到公开仓库。
